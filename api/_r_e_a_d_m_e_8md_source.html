1 # Arrowhead C++ library
2 
3 This is a C++ library for interfacing with Arrowhead services.
4 
5 
6 This project is very much a work in progress and the API is still subject to
7 major changes on a daily basis. The library is not ready for mainstream usage.
8 
9 The library relies on C++11 features, such as range-based for and auto variables
10 for developer convenience and increased productivity.
11 
12 ## Getting started
13 
14 The build system uses CMake
15 
16 ### Dependencies
17 
18 The following third party libraries are used internally:
19 
20  - CMake, for generating build system files (Unix Makefiles, Visual Studio files etc.)
21  - libcurl, for HTTP communication with the Arrowhead service directory REST API
22  - PugiXML, for XML parsing of results (optional)
23  - Doxygen, for generating API documentation (optional)
24  - log4cplus, for formatted debug output (optional)
25  - C++ standard library
26  - (bundled header-only, no install required) Catch C++ unit testing library
27 
28 Planned future deps:
29 
30  - libcoap? for interfacing with CoAP services
31  - ldns? for creating DNS-SD update requests
32  - openssl? for signing DNS-SD updates and for encrypted communication
33  - Some JSON lib? for speaking with JSON services
